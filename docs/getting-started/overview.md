# Getting started

This section gets you from **zero → running** on your machine.

## What you will run

- **FastAPI backend**: receives your query and runs the LangGraph agent (default port **8000**).
- **Streamlit frontend**: a chat-style UI that calls the backend (default port **8501**).

## Prerequisites

- **Python**: 3.12+ (required by `pyproject.toml`)
- API keys (see [`configuration.md`](configuration.md))

## Next steps

1. Install dependencies: [`installation.md`](installation.md)
2. Configure `.env` and providers: [`configuration.md`](configuration.md)
3. Run backend + UI: [`running-locally.md`](running-locally.md)
